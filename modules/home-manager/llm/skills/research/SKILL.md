---
name: research
description: Investigate a question against high-trust primary sources and report the findings with citations. Use when the user wants a topic researched, docs or API facts gathered, or reading legwork delegated to a background agent.
---

Spin up a **background agent** to do the research, so you keep working while it reads.

Its job:

1. Investigate the question against **primary sources** — official docs, source code, specs, first-party APIs — not a secondary write-up of them. Follow every claim back to the source that owns it.
2. Where authoritative sources are silent, say so explicitly — "no official guidance" is itself a finding.
3. **Report the findings back in the conversation** — a structured summary with a citation (URL or source path) per claim. Write NO files: not in the repo, not in the wiki, nowhere. The findings inform the task at hand; anything worth persisting is the user's call, afterwards.
